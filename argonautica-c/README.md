# argonautica-c

[argonautica-c](https://github.com/bcmyers/argonautica/tree/master/argonautica-c)
is wrapper for
[argonautica-rs](https://github.com/bcmyers/argonautica/tree/master/argonautica-rs).

It allows you to write C or C++ code that uses
[argonautica-rs](https://github.com/bcmyers/argonautica/tree/master/argonautica-rs).
